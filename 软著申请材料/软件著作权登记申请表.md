## 软件功能描述

| 项目 | 内容 |
|------|------|
| **开发目的** | 为无人机操控人员、航空器运营单位及低空空域管理机构提供广东省低空适飞区域在线智能查询服务，实现航线规划前的空域合规性预审，降低违规飞行风险 |
| **面向领域/行业** | 低空经济；无人机运营；航空安全；智慧城市 |

### 软件的主要功能

本软件具备以下核心功能：

1. **适飞区域航线穿越检查**：基于广东省 46,331 个适飞矩形网格数据，对用户提交的航线（不少于 2 个航路点）进行逐点采样判定，自动识别适飞区（绿）、受限区（红）、无数据区（灰），输出受限段列表及综合评级（PASS / WARNING / RESTRICTED）。

2. **CREAM 三维碰撞风险计算**：依据 ICAO Doc 9689 及新加坡南洋理工大学 ATMRI CREAM 方法论，计算低空无人机与有人驾驶航空器在纵向、侧向、垂直三个维度的碰撞风险值，支持同向/对向场景，采用 Laplace⊛Normal 解析卷积 TVE 分布。

3. **总垂直风险（TVR）计算**：叠加技术性高度误差（ASE+AAD）、高度层偏差事件（CLD）、错误高度层事件（WL）三种 LHD 情形，计算 Naz_total 总垂直风险值。

4. **地形 CFIT 风险分析**：调用 OpenTopoData SRTM 30m 数字高程模型 API，沿航线等距采样并计算每点超障余度（clearance）、CFIT 碰撞概率及最低安全高度（MSA），输出地形剖面与风险评级（PASS / WARNING / FAIL）。

5. **热点聚类检测**：实时拉取 adsb.fi ADS-B 数据，检测航迹交叉点，应用 K-means++ 聚类算法识别高密度碰撞风险热点区域，可视化输出聚类中心与权重。

6. **航路网络分析**：基于 NetworkX 图计算框架，对广东大湾区空域航路结构进行边/节点介数中心性分析、网络渗流分析，识别关键航段及潜在薄弱环节。

7. **空域复杂度计算**：采用 Prandini 等人的 Interacting Particle System 方法，在自定义网格内计算碰撞概率密度分布，量化评估局部空域复杂程度。

8. **实时气象查询**：接入 OpenWeatherMap API，获取大湾区各监测点实时气象数据（风速/风向/能见度），自动推算 CREAM 参数建议值（侧向速率 Vy、导航精度 RNP）。

9. **用户注册与登录**：提供手机号+密码注册/登录体系，密码采用 PBKDF2-HMAC-SHA256（迭代 260,000 次）安全哈希存储，Session Token 有效期 30 天，数据持久化至 Supabase。

10. **评估历史管理**：登录用户可查询本人历史评估记录（航线起终点、风险等级、时间戳），支持分页查询；管理员可查看全平台统计摘要。

11. **接口速率限制**：内置进程级 IP 速率限制器，计算类接口每 IP 每分钟最多 30 次请求，防止滥用与 DoS 攻击。

### 软件的技术特点

- 采用 FastAPI 异步 Web 框架，支持高并发请求处理，API 文档自动生成（/docs）。
- 空域网格数据采用 0.1° 桶分组空间索引，查询复杂度由 O(46,331) 优化至 O(1)，毫秒级响应。
- CREAM 碰撞风险使用 SciPy 高精度数值积分，精度优于前端 JavaScript 近似计算。
- TVE 分布采用 Laplace ⊛ Normal 解析卷积，数值稳定，无溢出风险。
- 地形高程查询具备内存 LRU 缓存（最多 200 批次），避免重复外部 API 调用。
- 全系统 CORS 白名单管理，安全限制跨域访问来源。
